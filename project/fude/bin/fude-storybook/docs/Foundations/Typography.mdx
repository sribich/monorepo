# Typography

## Fonts

The font is customizable through themeing... Should choose whether to focus on sans or serif font.

What do we use?

### How do you use it?

We recommend obtaining fonts through a CDN to give control of the themeing to the consumer.

<link
    href="https://fonts.googleapis.com/css2?family=Figtree:ital,wght@0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,300;1,400;1,500;1,600;1,700;1,800;1,900&family=Poppins:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap"
    rel="stylesheet"
/>

Download available https://fonts.google.com/specimen/Figtree

### Weights

We opt to use as few weights as possible in order to keep total resource sizes lean.

Light
Normal
Bold

### Considerations and fallbacks

Make sure we have fallbacks, especially for unicode. Roboto, etc Noto,

--font-family: Figtree, Roboto, Noto Sans Hebrew, Noto Kufi Arabic, Noto Sans JP, sans-serif;
--title-font-family: Poppins, Roboto, Noto Sans Hebrew, Noto Kufi Arabic, Noto Sans JP, sans-serif;
